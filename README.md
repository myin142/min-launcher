## Min Launcher

Features
- [x] Apps List + Toolbar
- [x] Search for installed apps
- [x] Show time + date
- [x] Show calendar events
- [x] ToDo list

Swipes

- Swipe up: Open apps list
- Swipe down: Open settings
- Swipe left/right: Open todo edit
